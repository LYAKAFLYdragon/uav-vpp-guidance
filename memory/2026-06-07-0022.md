## 关键决策:
1. **零方差边界处理**: `paired_t_test` 在 `std_diff ≈ 0` 时:
   - `mean_diff ≈ 0` → `t=0.0, p=1.0`（无显著差异）
   - `mean_diff ≠ 0` → `t=±inf, p=0.0`（差异极其显著）
2. **Cohen's d 零方差**: `std_diff ≈ 0` 时:
   - `mean_diff ≈ 0` → `d=0.0, magnitude='negligible'`
   - `mean_diff ≠ 0` → `d=±inf, magnitude='large'`
3. **类型安全**: 所有统计函数返回的布尔值统一用 `bool()` 转换为原生 Python `bool`，避免 `numpy.bool_` 与 `is` 运算符不兼容的问题。

## 遇到的问题及解决:
1. **11 个测试失败**: 根因是 scipy `ttest_rel` 在零方差时返回 nan，以及 `numpy.bool_` 与 Python `bool` 的 `is` 比较行为不同。
2. **effect-size 测试数据设计缺陷**: 原测试使用 `b = [x + c for x in a]` 产生理论上恒定的差异，`np.std` 因浮点舍入得到 `~4e-17`，导致 Cohen's d 高达 `~10^15`。修复为 `b = a[i] + c + noise[i]`，使差异具有适当方差。
