## 遇到的问题及解决:
1. **Include 路径解析错误**: `_resolve_config` 使用 `config_path.parent.parent` 导致 `config/config/guidance.yaml`。修复为 `config_path.parent.parent.parent`（因为 config 文件在 `config/experiment/` 下，项目根目录是上两级）。
2. **dry-run 时 checkpoint 强制要求**: `proposed_bilevel.yaml` 没有 `policy_checkpoint`。修复为 dry-run 模式下不强制要求 checkpoint。
3. **pandas tabulate 版本问题**: `DataFrame.to_markdown()` 需要 tabulate>=0.9.0，环境只有 0.8.10。修复为手动构建 Markdown 表格字符串。
