### 2.3 统计方法
- **Success Rate**: Wilson score interval (95%)
- **Continuous metrics**: percentile bootstrap CI (10,000 resamples)
- **Paired comparison**: Wilcoxon signed-rank test on (scenario, seed) matched pairs
- **Effect size**: Cohen's d for continuous, Cohen's h for proportions
- **约束**: 所有 p-value 必须同时报告 effect size；summary 中明确标注 preliminary vs paper-safe

### 2.4 Failure Taxonomy
优先级从高到低：
1. timeout / crash / out_of_bounds (显式终止原因)
2. miss (final_range_m > 200m)
3. candidate_tail_chase (scenario 以 candidate_ 开头)
4. prediction_degradation (cv/ca 且 mean_prediction_error_m > 50)
5. unstable_command (nz_cmd_modification_rate > 0.9)
6. low_energy_geometry (energy_proxy < 5000)
7. unclassified (兜底，不允许 silent)

本次 320 episodes 中 110 次失败，全部分类到明确原因，无 unclassified。

## 3. 关键发现

### 3.1 Overall Success Rate (Wilson 95% CI)
| Method | SR | CI |
|---|---|---|
| no_prediction | 75.00% | [64.52%, 83.19%] |
| cv_prediction | 62.50% | [51.55%, 72.31%] |
| ca_prediction | 62.50% | [51.55%, 72.31%] |
| gain_only | 62.50% | [51.55%, 72.31%] |

### 3.2 Paired Comparison vs Baseline (no_prediction)
- **cv/ca vs no_prediction**: return 显著更低 (p<0.001, d≈-0.37), min_range 显著更低 (d≈-0.49), nz_cmd_mean 差异大 (d≈-5.4)
- **gain_only vs no_prediction**: return 显著更低 (p<0.001, d≈-0.29), min_range 显著更低 (d≈-0.89)

### 3.3 Stratified
- **Regression**: no_prediction 100%, cv/ca 75%, gain_only 100%
- **Candidate**: no_prediction 50%, cv/ca 50%, gain_only 25%

### 3.4 Claims 评估

| Claim | 状态 | 理由 |
|---|---|---|
| Neural prediction (cv/ca) improves feasible-geometry tracking | ❌ Not paper-safe | 整体 SR 低于 no_prediction；paired return 显著更差 |
| CEM-optimized gains (gain_only) improve over baseline | ❌ Not paper-safe | 整体 SR 持平或更低；candidate 场景表现更差 |
| no_prediction baseline is strongest on regression suite | ✅ Paper-safe (limited scope) | regression 100% SR，cross-seed 一致 |
| Tail-chase candidates remain challenging for all methods | ✅ Paper-safe | candidate_head_on_far/medium 全方法 0% |
| Mode-switch / pure-PN claims require Stage 6G.5D data | ✅ Paper-safe (already established) | 不在本 benchmark 矩阵内，但此前已验证 |

> **注意**: 本次 Stage 9B.1 结果与 Stage 6F/6G 的发现一致——在标准回归场景下预测有一定作用，但在 candidate tail-chase 场景下所有 VPP-based 方法均失败。cv/ca 在此 simple backend 上并未表现出优势，可能与 predictor 冻结或场景特性有关。
